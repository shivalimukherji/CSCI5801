\doxysubsection{Detailed Description}
The voting system to conduct the election and determine a winning candidate. 

to distribute the votes, the PO Vote System will use the ballot processing system associating with it.

In the case of a tie, the PO Vote System will run the \mbox{\hyperlink{classTieBreaker}{Tie\+Breaker}} process and a winner will be chosen
