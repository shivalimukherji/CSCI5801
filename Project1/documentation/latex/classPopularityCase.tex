\doxysubsection{Detailed Description}
A class representing a popularity special case implementation. 

This class inherits from the \mbox{\hyperlink{classISpecialCase}{I\+Special\+Case}} interface and provides an implementation for the \mbox{\hyperlink{classPopularityCase_a35aab4def42922f9afb108827a42a86a}{run()}} method. It takes a vector of \mbox{\hyperlink{classIRCandidate}{I\+R\+Candidate}} pointers as input and stores it in a private member variable. It also provides a \mbox{\hyperlink{classPopularityCase_a33c70a42d6dcc2edafe1d855a86e9f25}{helper()}} method and getter/setter methods for the candidates member variable.
