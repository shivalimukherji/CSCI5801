The abstract class where all Vote\+Ballots are inherited from. 

Vote\+Ballots contains all necessarily information and is produced by a \mbox{\hyperlink{classBallot}{Ballot}} Processing class.

Vote\+Ballots are used in its pertaining Vote Election to conduct an election
