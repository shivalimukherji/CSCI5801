\doxysubsection{Detailed Description}
IR Processing is to process all ballots in a .csv file and prepare an organized set of information to send to the \mbox{\hyperlink{classIRVoteSystem}{I\+R\+Vote\+System}} class, conducting and determining the winner. 

T\+He processing will also handle redistribution if a candidate is eliminated.
